# 券包管理页面实施计划

> **目标：** 在礼券主题模块下新增 `券包管理` 菜单，并实现券包管理列表页与新增/编辑弹窗原型。
> **架构：** 基于现有礼券主题模块的列表页、通用弹窗、抽屉与表格样式进行复用。通过新增独立路由、独立前端数据集和弹窗草稿对象，实现“列表页 + 长弹窗 + 券明细表格”的原型交互，并统一将截图里的 `活动` 文案替换为 `券包`。
