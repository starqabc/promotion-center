# 左侧导航重组实施计划

> **目标：** 按新的业务分组重组左侧导航，将现有菜单拆分为 `促销规划 / 促销模版 / 促销活动 / 促销调度 / 促销计算 / 主档信息 / 促销监控` 等一级菜单。
> **架构：** 通过修改 `index.html` 中的静态一级菜单结构，配合 `app.js` 中的路由归属、高亮规则和 `campaignsSubNav` 动态子导航渲染来完成导航重组。业务页面路由和主体逻辑不变，仅调整菜单层级和展示位置。

---

**修改文件：**

- `index.html`
  - 负责左侧一级菜单和普通二级菜单结构
- `app.js`
  - 负责导航高亮、路由归属、促销活动子导航动态渲染

- `docs/superpowers/specs/2026-06-13-nav-restructure-design.md`
