# 促销调度终止单实施计划

> **目标：** 精简 `生效商品` 列表页，并在 `促销调度` 下新增独立的 `促销终止单` 列表、新增、详情原型页面。
> **架构：** 通过修改 `index.html` 增加菜单入口，在 `app.js` 中收口 `生效商品` 列表展示能力，并新增一组 `promo-terminate-orders*` 路由、页面状态、样例数据、页面渲染与事件处理。继续复用现有查询区、表格、整页表单和详情展示模式，必要时仅补少量样式。

---

## 文件结构映射

**修改文件：**

- `index.html`
  - 负责 `促销调度` 二级菜单入口
- `app.js`
  - 负责 `生效商品` 列表精简
  - 负责 `促销终止单` 路由标题、样例数据、查询状态
  - 负责 `促销终止单` 列表 / 新增 / 详情渲染与事件
- `styles.css`
  - 仅在新增页布局与现有样式不匹配时补充少量样式

**设计文档：**

- `docs/superpowers/specs/2026-06-13-dispatch-terminate-order-design.md`
  - 记录本轮促销调度调整范围与页面口径

**计划文档：**
