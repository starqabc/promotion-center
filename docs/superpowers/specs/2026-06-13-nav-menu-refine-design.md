# 导航菜单文案与层级微调设计

## 1. 背景

本次仅继续调整左侧导航，不修改业务页面内部字段、表单、列表和流程逻辑。

项目约束保持不变：

- 仅 HTML / CSS / JS
- 无依赖
- 支持 `file://` 直接打开
- 左侧菜单 + Hash 路由
- 只改本轮提到的菜单和层级

用户本轮需求：

1. `选品汇总` 改为 `档期商品`
2. `模版优先级` 改为 `模版规则设置`
3. `促销商品终止` 改为 `生效商品`
4. 一级菜单 `促销活动` 下去掉中间层二级菜单 `促销活动`
5. 其余现有活动分组升级为 `促销活动` 下的直接二级结构

用户已确认本轮采用：

- `分组升二级`

## 2. 目标

### 2.1 菜单文案统一

需要统一修改以下名称：

- `选品汇总` -> `档期商品`
- `模版优先级` -> `模版规则设置`
- `促销商品终止` -> `生效商品`

### 2.2 促销活动层级上移

调整后，一级菜单 `促销活动` 下不再出现中间层按钮 `促销活动`。

调整后的二级结构为：

1. 档期智能促销
2. 特价
3. 折扣
4. 满减满赠
5. 满赠券促销

其中：

- `档期智能促销` 为普通二级入口
- `特价 / 折扣 / 满减满赠 / 满赠券促销` 继续保留现有展开式分组
- 各分组下的具体玩法仍保持当前动态生成逻辑

## 3. 非目标

- 不新增页面
- 不新增或修改路由地址
- 不调整任一业务页面内部内容
- 不把具体玩法直接提升为二级菜单
- 不改其他一级菜单结构

## 4. 现状分析

### 4.1 静态菜单文案

`index.html` 中当前仍存在以下旧文案：

- `选品汇总`
- `模版优先级`
- `促销商品终止`

这些属于静态左侧菜单，需要直接改名。

### 4.2 动态促销活动子导航

当前 `促销活动` 的结构不是普通二级菜单，而是：

1. `index.html` 中存在一个中间层折叠按钮 `促销活动`
2. `app.js` 中通过 `renderCampaignSubNav()` 渲染其内部内容
3. 内部内容包含：
   - `档期智能促销`
   - `特价`
   - `折扣`
   - `满减满赠`
   - `满赠券促销`

因此本次不能只改 `index.html`，还需要同步调整 `app.js` 中的导航渲染和高亮映射。

## 5. 方案选择

本次采用方案 A：`分组升二级`。

### 5.1 备选方案

- 方案 A：分组升二级
- 方案 B：所有玩法升二级
- 方案 C：仅改文案，不动层级

### 5.2 选择原因

- 与用户确认的口径一致
- 改动范围最小，只动导航层
- 保留现有活动分组展开逻辑，回归风险更低

## 6. 详细设计

### 6.1 `促销规划` 下文案调整

- 左侧菜单 `选品汇总` 改为 `档期商品`
- 对应路由 `/assortment`、`/assortment-summary` 的页面标题、面包屑文案同步改为 `档期商品`
- `创建档期选品` 等编辑页标题可继续保留原业务语义，不强制改为 `档期商品`

### 6.2 `促销模版` 下文案调整

- 左侧菜单 `模版优先级` 改为 `模版规则设置`
- 对应路由 `/template-priority`、`/template-priority-create`、`/template-priority-edit` 的标题和副标题同步替换
- 路由地址仍保持 `/template-priority*`

### 6.3 `促销调度` 下文案调整

- 左侧菜单 `促销商品终止` 改为 `生效商品`
- 对应路由 `/promo-goods-terminate` 的标题和副标题同步替换
- 页面主体逻辑不变，仍复用现有原型页面

### 6.4 `促销活动` 层级调整

当前结构：

- 一级：`促销活动`
- 二级：`促销活动`
- 三级：`档期智能促销 / 特价 / 折扣 / 满减满赠 / 满赠券促销`

调整后结构：

- 一级：`促销活动`
- 二级：`档期智能促销 / 特价 / 折扣 / 满减满赠 / 满赠券促销`

实现口径：

- 删除中间层折叠按钮 `促销活动`
- `campaignsSubNav` 直接挂在 `促销活动` 一级菜单面板下
- `档期智能促销` 继续作为普通二级入口展示在顶部
- 四个活动分组继续通过 `.nav__subgroup` 折叠展开
- 分组内具体玩法仍使用 `/campaigns-pt/<promoType>` 路由

### 6.5 高亮与展开逻辑

需要同步保持以下行为：

- 进入 `/campaigns*` 或 `/template-priority-schedule-import*` 时，高亮一级菜单 `促销活动`
- 进入 `/template-priority-schedule-import*` 时，高亮二级入口 `档期智能促销`
- 进入 `/campaigns-pt/*`、`/campaigns-detail/*`、`/campaigns-create`、`/campaigns-edit` 时，自动展开所属活动分组
- 移除中间层 `促销活动` 后，不再依赖 `campaignsGroup` 的展开态
